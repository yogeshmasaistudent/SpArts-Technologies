# SpArts Technologies Assigment 

This project is a web application that helps parents find the best activities for their children. The application features a hero section, a content section showcasing top academies, a testimonials section with feedback from parents, and a subscription form in the footer.

## Features
- **Hero Section**: A welcoming section with an image and buttons for academies and in-home activities.
- **Content Section**: A slider showcasing top academies with images.
- **Testimonials Section**: A slider with testimonials from satisfied parents.
- **Footer**: A subscription form for users to stay updated.

## Technologies Used
- React
- Tailwind CSS
- Slick Carousel
- Flexbox
- Responsive Design

## Setup and Installation
To get started with the project, follow these steps:

1. **Clone the repository**
    ```bash
    git clone https://github.com/yogeshmasaistudent/SpArts-Technologies.git
    cd frontend
    ```

2. **Install dependencies**
    ```bash
    npm install
    ```

3. **Run the application**
    ```bash
    npm start
    ```

## Usage
To use the application:

1. Open your browser and go to link see our website . 
2. Navigate through the sections to explore various features.
3. Use the subscription form to stay updated.

## Components

### HeroSection
Displays a hero image with introductory text and buttons.

### ContentSection
Showcases top academies using a slider.

### TestimonialsSection
Contains a slider with testimonials from parents.

### Footer
Provides a subscription form for users to stay updated.
